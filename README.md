<hr/>

<h2>Funcionalidades</h2>
<ul>
  <li>Tabuleiro completo em HTML/CSS estilizado com Bootstrap</li>
  <li>Movimentos das peças implementados em JavaScript (ES6+) e jQuery</li>
  <li>Validação de jogadas conforme regras oficiais</li>
  <li>Interface responsiva para desktop e mobile</li>
</ul>

<h2>Instalação e Uso</h2>
<ol>
  <li>Clone o repositório:
    <pre><code>git clone https://github.com/renan00923/chess-js-game.git</code></pre>
  </li>
  <li>Acesse a pasta:
    <pre><code>cd chess-js-game</code></pre>
  </li>
  <li>Abra o arquivo <code>index.html</code> no navegador.</li>
</ol>

<p>Ou rode um servidor local simples:</p>
<pre><code>npx serve</code></pre>
<p>Depois acesse <code>http://localhost:3000</code></p>
