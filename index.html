Sobre

O ChessJS é um projeto feito para apaixonados por xadrez. Nosso objetivo é proporcionar uma experiência divertida, intuitiva e acessível, seja para iniciantes ou jogadores avançados com um jogo simples de xadrez.

Desenvolvido em HTML, CSS e JavaScript, o site oferece partidas contra a máquina, modo multiplayer local e dicas para você evoluir no jogo.

Sinta-se à vontade para sugerir melhorias ou reportar bugs. Bom jogo!
